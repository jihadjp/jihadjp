## About

I'm a software developer based in Bangladesh with **3+ years of experience** building production-grade Android and web applications. My focus is on writing clean, maintainable Java code — from mobile UIs to backend APIs.

I started out curious about how digital products are built. That curiosity became a career. Today I work across the full stack, from Android frontends to Spring Boot services and MySQL-backed web applications.

I also run coding workshops and contribute to local tech communities — because the best way to grow is to teach.
